ATIVIDADE: Utilize o scikit-fuzzy para representar algum problema de seu interesse e permitir que perguntas sejam realizadas e respondidas usando inferência difusa.

### Programas utilizados ###

- O projeto foi desenvolvido em máquinas com Linux UBUNTU 16.04 LTS, feito em Python 3 e usando as bibliotecas Skfuzzy, Numpy e MatplotLib.

### Instalação da Bibliotecas ###

 - Para instalar as bibliotecas, abra o terminal e escreva os comandos abaixo, um de cada vez:

 > sudo apt install python3-pip
 > pip3 install --user networkx==2.3
 > pip3 install --user scikit-fuzzy
 > pip3 install --user matplotlib
 > pip3 install --user numpy

### Como utilizar ###

 - Para utilizar o programa corretamente é preciso certificar que todas as bibliotecas necessárias estejam instaladas;
 - Extraia o arquivo para a pasta desejada;
 - Abra o terminal e acesse o diretório extraído;
 - Execute o arquivo do programa pelo comando 'python Logica_Fuzzy.py';
 - Uma janela irá se abrir contendo os campos onde será inseridos os valores e botões que realizam operações de cálculo e exibição dos gráficos;
 - Para finalizar o programa, feche todas as janelas abertas ou digite o comando "Ctrl+C" no terminal;

### Funcionamento ###

 - O programa utiliza entradas como velocidade e limite (de velocidade) para calcular o valor da multa, que varia com as entradas e a diferença entre elas.
 - Caso a velocidade seja menor ou igual ao limite não será aplicada multa, apenas uma mensagem de "sem multa" e "advertência".
 - Os gráficos exibem as funções de partinência e o resultado das entradas inseridas.
